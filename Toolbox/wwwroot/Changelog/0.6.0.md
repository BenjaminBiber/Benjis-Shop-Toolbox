#### 🆕 Anpassen der Extension-Version

- **Im Aktionen-Menü der Extension-Seite (Schraubenschlüssel) kann man nun die Version aller Projekte innerhalb einer Extension upgraden oder ändern.**
- **Im Aktionen-Menü der Extension-Seite (Schraubenschlüssel) kann man nun, falls vorhanden, den Changelog der Extension einsehen.**
- **Die Extensions-Tabelle zeigt zudem die aktuelle Version der Extension an.**
